subs

Subscription manager.

A fast and uniform way of following accounts on video platforms without an account, and without going through the platform's interface and notification system (especially useful to avoid YouTube's web interface and conspicuously forgetful notifications). Currently handles:

  • LBRY, using the JSON API of a local lbrynet server
  • YouTube, using yt-dlp

Video history is kept in a local SQlite database and presented via either a CLI or a curses TUI interface. The application is programmable and extendable using Lua.
