Below you'll find other means to contact me, in descending order of preference. You'll also find links to social media I reluctantly use. Those for the most part serve just to publish inferior versions of the pages here, with links to them.

I highly recommend — not just for this web site, but for everything on the internet — using the RSS feed (which you can also find via the icon on the top right of every page) to be notified when I publish something new.

You can find the code behind this web site here. It's very simple HTML, with a minimal amount of CSS and Javascript, all written from scratch and served statically by one nginx server hosted in Hetzner.

For almost a decade it was all manually written using only vim. Then, I created a small static web site generator, also written from scratch in Lua, because I'm a programmer and that's what we do for recreation. You can find all of the code and the history in the git repository linked above.
